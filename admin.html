서버에 글 올리기

브라우저에만 저장된 글을 JSON 파일로보내 서버에 함께 배포합니다.

왜 4번 글이 서버에 안 보이나요?

지금 글쓰기는 내 컴퓨터 브라우저(localStorage)에만 저장됩니다. HTML·CSS만 서버에 올리면 글 데이터는 따라가지 않습니다.

  1. 아래에서 JSON보내기로 blog.json / board.json 받기
  2. 프로젝트의 data/ 폴더에 같은 이름으로 덮어쓰기
  3. data 폴더까지 포함해 호스팅에 다시 업로드

방문자 모두에게 보이게 됩니다. (게시판 번호 4는 목록 순서일 뿐, JSON에 글이 있으면 표시됩니다.)

블로그

현재 저장된 블로그 글을 data/blog.json 형식으로 받습니다.

게시판

현재 저장된 게시글을 data/board.json 형식으로 받습니다.

장기적으로는 DB 연동(예: Supabase, Firebase)을 쓰면 글쓰기 즉시 서버 반영이 가능합니다.
